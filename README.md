# Valorant Agent Recommendation System

A machine learning project using K-Nearest Neighbor (KNN) to recommend optimal agents based on map conditions.

## 🔍 Features
- Input map → get recommended agents
- Uses KNN algorithm
- Simple dataset simulation

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas

## 🎯 Purpose
To demonstrate how machine learning can support decision-making using data patterns.

## 📊 Result
Provides structured recommendations based on similarity calculation.
